# Project autonomy

Codex may find bounded bugs, shape issues, implement and test, push branches, open pull requests, and merge after green checks and independent review. Preserve existing content and data contracts.

Production deploy/restart, stored user content, secrets, destructive changes, broad product/visual direction, or URL/base-path migration require Henry. Use Terra high for ambiguous multi-file changes and Sol high for security, architecture, migrations, or independent review. Model strength never expands A3.
