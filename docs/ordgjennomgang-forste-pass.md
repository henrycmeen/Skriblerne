# Første-pass ordgjennomgang

Dette er en avgrenset startliste for den manuelle Henry/Ellinor-gjennomgangen. Listen er ikke fasit. Den peker bare på ord som kan være abstrakte, lite bildevennlige, veldig spesifikke, litt vanskelige eller mindre passende for en barne-/familieapp.

Lag en importbar JSON med:

```bash
npm run review:first-pass -- --output /tmp/skriblerne-forste-pass.json
```

Importer filen i `ordliste.html`, åpne filteret `Se på`, og avgjør om forslagene skal beholdes, endres eller forkastes.
